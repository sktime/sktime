# Contributing to sktime

Thank you for your interest in contributing to sktime!

## Development Setup

1. Clone your fork of the repository.

2. Install the package in editable mode with development dependencies:

   pip install -e ".[dev]"

3. Install pre-commit hooks:

   python -m pre_commit install

This ensures that formatting, linting, and other checks run automatically before each commit.

You can find additional contribution guidelines on our website:
https://www.sktime.net/en/latest/get_involved/contributing.html
