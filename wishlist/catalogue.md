# catalogue Wishlist

Wishlist of catalogues (collections of objects, datasets, estimators, etc.) requested for implementation in sktime.

## How to Add Items

1. Open an issue on the [sktime issue tracker](https://github.com/sktime/sktime/issues) describing the catalogue
2. Add the item below following the format template
3. Submit a PR with your addition

## Format Template

```markdown
- [ ] **Catalogue Name** - Brief description
  - Reference: [Link](link)
  - Issue: #1234
```

## Wishlist Items

- [ ] **Model zoo/registry systems** - Central registries of estimators or pipelines, discoverable and queryable from sktime.
- [ ] **AutoML catalogues** - Catalogues of AutoML workflows, search spaces, and configurations for time series tasks.
- [ ] **Benchmark suite managers** - Tools to define, store, and reuse collections of datasets, tasks, and evaluation protocols.
- [ ] **Algorithm discovery/search tools** - Catalogues that support searching for compatible algorithms given task metadata.
- [ ] **Hyperparameter configuration catalogues** - Libraries of tuned hyperparameter configurations for common algorithms and datasets.
- [ ] **Pre-trained model repositories** - Catalogues of pre-trained time series models ready for fine-tuning or zero-shot use.
- [ ] **Ensemble catalogues** - Registries of reusable ensemble definitions and recipes.
- [ ] **Plugin/extension registries** - Catalogues of third-party sktime extensions, plugins, and external algorithm wrappers.
