# catalogue Wishlist

Wishlist of catalogues (collections of objects, datasets, estimators, etc.) requested for implementation in sktime.

## How to Add Items

1. Open an issue on the [sktime issue tracker](https://github.com/sktime/sktime/issues) describing the catalogue
2. Add the item below following the format template
3. Submit a PR with your addition

## Format Template

```markdown
- [ ] **Catalogue Name** - Brief description
  - Reference: [Link](link)
  - Issue: #1234
```
