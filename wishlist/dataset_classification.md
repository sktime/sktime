# dataset_classification Wishlist

Wishlist of classification datasets requested for inclusion in sktime.

## How to Add Items

1. Open an issue on the [sktime issue tracker](https://github.com/sktime/sktime/issues) describing the dataset
2. Add the item below following the format template
3. Submit a PR with your addition

## Format Template

```markdown
- [ ] **Dataset Name** - Brief description
  - Source: [Link](link)
  - Issue: #1234
```

## Wishlist Items

- [ ] **UCR Time Series Archive datasets (128+ individual datasets)** - Additional UCR datasets not yet wrapped in sktime.
- [ ] **UEA Multivariate Archive datasets (30+ datasets)** - Multivariate classification benchmarks with ready-made loaders.
- [ ] **Audio classification datasets (UrbanSound, ESC-50, AudioSet)** - Time series/audio classification benchmarks.
- [ ] **HAR (Human Activity Recognition) datasets (PAMAP2, OPPORTUNITY, WISDM)** - Wearable sensor based human activity recognition datasets.
- [ ] **ECG/EEG classification datasets (MIT-BIH, PhysioNet, EEG Motor Movement)** - Physiological signal classification datasets.
- [ ] **Gesture recognition datasets (Kinect, Leap Motion)** - Time series of motion/pose for gesture recognition.
- [ ] **Motion capture datasets** - Skeleton and motion capture sequences for action/gesture classification.
- [ ] **Sensor signal datasets** - Multimodal sensor recordings for industrial or consumer devices.
- [ ] **Medical time series classification** - Clinical time series datasets for diagnostic classification.
- [ ] **Spectral data classification** - Spectroscopy and spectral time series classification tasks.
- [ ] **Seismic signal classification** - Earthquake and seismic event time series datasets.
- [ ] **Industrial process monitoring datasets** - Fault detection and state classification benchmarks.
- [ ] **Biometric time series** - Gait, keystrokes, and other biometric sequence datasets.
- [ ] **Speech/phoneme classification** - Time series view of speech and phoneme recognition benchmarks.
