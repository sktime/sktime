# dataset_regression Wishlist

Wishlist of regression datasets requested for inclusion in sktime.

## How to Add Items

1. Open an issue on the [sktime issue tracker](https://github.com/sktime/sktime/issues) describing the dataset
2. Add the item below following the format template
3. Submit a PR with your addition

## Format Template

```markdown
- [ ] **Dataset Name** - Brief description
  - Source: [Link](link)
  - Issue: #1234
```

## Wishlist Items

- [ ] **Time series regression benchmark datasets** - Benchmarks where the target is a continuous-valued series or scalar.
- [ ] **Exogenous variable prediction datasets** - Datasets focused on predicting exogenous or covariate series.
- [ ] **Transfer function datasets** - Time series regression with transfer function/ARIMAX style relationships.
- [ ] **Causal inference time series datasets** - Intervention and treatment series with causal ground truth.
- [ ] **Treatment effect datasets** - Time series exposure/treatment datasets for heterogeneous effect modeling.
- [ ] **Counterfactual time series** - Benchmarks for counterfactual forecasting and policy analysis.
- [ ] **Multi-output regression datasets** - Datasets with multiple related regression targets over time.
