# Network Wishlist

Wishlist of deep learning network architectures requested for implementation in sktime.

## How to Add Items

1. Open an issue on the [sktime issue tracker](https://github.com/sktime/sktime/issues) describing the network
2. Add the item below following the format template
3. Submit a PR with your addition

## Format Template

```markdown
- [ ] **Network Name** - Brief description
  - Paper: [Title](link) or Reference: [Package](link)
  - Issue: #1234
```
